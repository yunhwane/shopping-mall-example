### Issue tracker

Issues and PRDs live as GitHub issues in `yunhwane/shopping-mall-example`, managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
